# 💰 QuickSave

A fintech application for managing savings, built as a full-stack mobile and backend system.

![QuickSave Preview](./quicksave-mobile/assets/preview.png)

## Overview

QuickSave provides users with a streamlined way to manage personal savings through a dedicated mobile interface backed by a secure, scalable API.

## Features

- 💵 Track and manage savings activity
- 🔐 Secure backend API for handling financial data
- 📱 Dedicated mobile client built for a smooth user experience
- 🏗️ Separated backend and mobile architecture for independent scaling

## Tech Stack

- **Language:** TypeScript
- **Mobile:** React Native
- **Backend:** Node.js

## Project Structure
quicksave/

├── quicksave-backend/   # API and business logic

└── quicksave-mobile/    # React Native mobile client

## Getting Started

**Backend:**
```bash
cd quicksave-backend
npm install
npm run dev
```

**Mobile:**
```bash
cd quicksave-mobile
npm install
npx expo start
```
